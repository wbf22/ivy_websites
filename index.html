Not every website can be hosted without on going costs. Unfortunately, certain features require what's called a server. This is a computer that stores data for your site. When someone visits your site in a browser like Chrome or Safari, the browser connects to your server to get that data. If your website needs one of the following then you'll need a server:
  • User accounts
  • Email sending
  • Payment processing
  • Order processing
  • Or basically anything other than an informational site

The features above typically require storing data which is why you need a dedicated server. This is typically a few dollars to hundreds of dollars a month, depending on how many visitors you have to your website. (only large companies will need expensive servers though. Most only need a server that costs about $4/month)

However, if you don't need to store data, there are options for hosting your website for free! We can make you a website with these options or convert an existing website for you!

For most businesses, a website usually only needs to help visitors understand provided services, provide contact and information, and display offers and deals; and this can be done without a dedicated server.
